在Linux中，可以使用以下命令设置路由优先级：
##### 1. 使用route命令设置路由优先级，例如：

`route add -net 192.168.0.0/24 gw 192.168.1.1 metric 100`

其中，metric参数表示路由的优先级，<font color=Red>**值越小优先级越高**</font>。
##### 2. 使用ip命令设置路由优先级，例如：

`ip route add 192.168.0.0/24 via 192.168.1.1 metric 100`

同样，metric参数表示路由的优先级，值越小优先级越高。
##### 3. 使用配置文件设置路由优先级，例如：
在/etc/network/interfaces文件中添加以下内容：

`up ip route add 192.168.0.0/24 via 192.168.1.1 metric 100`

这样，在系统启动时就会自动设置路由优先级。
需要注意的是，设置路由优先级时，需要考虑网络拓扑结构和路由策略，以确保网络通信的稳定和可靠。
